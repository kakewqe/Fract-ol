# 📄 Fract-ol

## 🇬🇧 English

The goal of this project is to explore mathematical fractals and graphical programming.

### Description

Fract-ol is a graphical project that renders fractals such as Mandelbrot and Julia sets.

This project helps you better understand:

- computer graphics
- mathematical visualization
- event handling
- complex numbers

### Program

```bash
./fractol mandelbrot
./fractol julia
```

### How it works

- Uses MiniLibX for rendering
- Draws fractals pixel by pixel
- Supports zooming and navigation
- Handles keyboard and mouse events

### Compilation

```bash
make
```

### Skills Developed

- C programming
- graphics programming
- event management
- mathematical algorithms
- optimization
- MiniLibX
